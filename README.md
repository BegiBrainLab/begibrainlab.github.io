# BegiBrainLab website

Static multilingual starter website for GitHub Pages.

Default language: English at `/`.
Spanish: `/es/`.
Basque: `/eu/`.

## Deploy on GitHub Pages
1. Create a public repository, e.g. `begibrainlab-web`.
2. Upload all files in this folder to the root of the repository.
3. Go to Settings > Pages.
4. Source: Deploy from branch; Branch: `main`; Folder: `/root`.
5. Keep the `CNAME` file for `www.begibrainlab.com`.

## Visual assets
Replace placeholder images by adding files to `assets/img/` and editing the hero/card sections in the HTML files.
